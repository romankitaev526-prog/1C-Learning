# 1C-Learning: Учебные алгоритмические задачи

Репозиторий содержит решения классических алгоритмических задач, реализованных на встроенном языке **1С:Предприятие 8.3**. Каждое решение представлено в виде внешней обработки (.epf) и текстового файла с исходным кодом (.bsl).

### Проекты в репозитории:

1. [**01_MissingNumber**](./01_MissingNumber/) — Поиск минимального отсутствующего положительного числа. 
   * *Навыки:* Массивы, Соответствия (Hash Map), обработка исключений.
2. [**02_CommonPrefix**](./02_CommonPrefix/) — Поиск самого длинного общего префикса в массиве строк.
   * *Навыки:* Работа со строковыми функциями, оптимизация вложенных циклов.
3. [**03_WaterVolume**](./03_WaterVolume/) — Расчет объема удерживаемой воды (Rain Water Trapping).
   * *Навыки:* Продвинутая алгоритмика, метод двух указателей, работа на стороне сервера.
